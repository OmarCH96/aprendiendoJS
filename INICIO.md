## 🚀 CÓMO USAR ESTE PROGRAMA

### PASO 1: Estructura de Cada Módulo

Cada módulo sigue este flujo de aprendizaje:

```
1. 📄 GUIA-RAPIDA.md
   └─ Referencia rápida (consulta cuando tengas dudas)

2. 📖 TEORIA.md
   └─ Lee esto primero (conceptos explicados)

3. ✅ EJEMPLO-RESUELTO.js
   └─ Ejecuta y estudia el código comentado

4. 🎯 EJERCICIOS-GUIADOS.js
   └─ Práctica con asistencia paso a paso

5. 💪 EJERCICIOS-INDEPENDIENTES.js
   └─ Resuelve por tu cuenta (¡aquí aprendes DE VERDAD!)

6. 📝 VERIFICACION.md
   └─ Evalúa tu comprensión antes de avanzar
```

### PASO 2: Metodología de Trabajo

```javascript
// REGLAS DE ORO:

1. ❌ NO avances al siguiente módulo hasta dominar el actual

3. 💬 PREGUNTA cuando algo no esté claro
4. 🔄 PRACTICA más allá de lo propuesto
5. 🧠 RELACIONA cada concepto con casos reales
```

### PASO 3: Ejecutar el Código

Para ejecutar los archivos JavaScript:

```powershell

# Ejemplo:

```

## 🎓 SISTEMA DE APRENDIZAJE

### Cuando estudies TEORÍA:
- Lee completamente sin apuros
- Intenta entender el "por qué", no solo el "qué"
- Relaciona con cosas que ya conoces
- Toma notas personales si ayuda

### Cuando veas EJEMPLOS:
- Ejecuta el código tú mismo
- Modifica valores para experimentar
- Lee los comentarios cuidadosamente
- Intenta predecir qué hará antes de ejecutarlo

### Cuando hagas EJERCICIOS GUIADOS:
- Sigue las pistas proporcionadas
- No copies y pegues sin entender
- Si algo falla, investiga por qué
- Pide aclaración si las pistas no bastan

### Cuando hagas EJERCICIOS INDEPENDIENTES:
- ⏱️ Intenta al menos 10 minutos por ejercicio
- 🚫 NO mires soluciones inmediatamente
- 🤔 Si te bloqueas, pide una PISTA, no la solución completa
- ✅ Cuando termines, solicita retroalimentación

## 💡 CÓMO PEDIR AYUDA

### ✅ Buenas formas de pedir ayuda:

```
"Estoy en el ejercicio 3 de variables. Intento hacer X pero obtengo Y.
Mi código es: [tu código]. ¿Qué estoy haciendo mal?"

"No entiendo qué significa 'scope anidado'. ¿Puedes explicarlo 
con otro ejemplo?"

"Completé el ejercicio de inventario. ¿Puedes revisar mi solución
y darme retroalimentación?"
```

### ❌ Evita:

```
"Dame la solución del ejercicio 4"
"No entiendo nada, explícamelo todo otra vez"
"¿Está bien esto? [sin contexto]"
```
